# VPN

VPN Overlay Network for Zencrypted.

## Goal

Build a VPN tunnel:

TUN <-> Erlang <-> UDP <-> Erlang <-> TUN

with X.509 PKI integration via synrc/ca.

## Milestones

1. TUN interface
2. Packet forwarding
3. UDP transport
4. Remote tunnel
5. PKI integration
6. User management
7. Security profile compliance
